#### 项目特点
* 通用性强，能够满足大多数行业的设备管理需要。
* 界面优雅；原本计划使用layuiAdmin，奈何对于业余开发者来说授权太贵且不开源，于是使用layui改写样式，高仿layuiAdmin的UI，自己集成了一套前端框架，保证代码全部开源。
* 部署简单，1期开发完成后会编写部署文档。
