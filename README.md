# GDMS-DeviceManagementSystem

* 基于web的设备管理系统，开发中，进展迅速，敬请期待~！
* 前后端完全分离，使用JWT管理session，后端C#只提供ajax接口，可通过简单的2次开发更换为任意后端语言
* 数据库使用了Oracle，开发完成后可能还会开发MySQL版
